# 【插件】移动端元素拖拽、缩放]

[etool](https://github.com/hewq/Front-end/blob/master/apps/JavaScript/elementtool/js/etool.js)

| 拖拽         | etool.drag(obj)        |
| ------------ | ---------------------- |
| obj.$dom     | 需要移动的 dom 节点    |
| obj.top      | 可移动范围的顶端临界点 |
| obj.right    | 可移动范围的右端临界点 |
| obj.bottom   | 可移动范围的下端临界点 |
| obj.left     | 可移动范围的左端临界点 |
| obj.callback | 回调函数               |

```javascript
etool.drag({
	$dom: $('.drag'),
	top: 0,
	right: 150,
	bottom: 150,
	left: 0,
	callback: function () {
		console.log(110);
	}
});
```

| 缩放         | etool.scale(obj)                            |
| ------------ | ------------------------------------------- |
| obj.$dom     | $dom 需要缩放的 dom 节点                    |
| obj.scale    | 可缩小的最小宽度                            |
| obj.minWidth | 可放大的最大宽度                            |
| obj.maxWidth | 手指每缩放 1px 实际元素的缩放大小；默认为 1 |
| obj.callback | 回调函数                                    |

```javascript
etool.scale({
	$dom: $('.scale'),
	minWidth: 100,
	maxWidth: 500,
	scale: 1,
	callback: function () {
		console.log(120);
	}
});
```
